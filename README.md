# Cloud IRC
Shred-down IRC chat in the cloud

# Technology stack
Bare-bones [node](http://nodejs.org/) & [express](http://expressjs.com/)

# Dev-Setup instructions
```
# Bootstrap
./bootstrap.sh

# Run
node app.js
```

# Test Online
